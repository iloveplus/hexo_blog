基于静态博客[Hexo](https://hexo.io/zh-cn/)，[NexT](http://theme-next.iissnan.com/)主题的博客。

## 添加新博客
`hexo new [layout] 文章标题`

## 本地运行
`npm run start`  

## 编译部署
`npm run publish`

## 添加主题 
`cd your-hexo-site 
$ git clone https://github.com/iissnan/hexo-theme-next themes/next
修改_config.yml 里面 theme: next`
